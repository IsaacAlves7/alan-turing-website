Somador

Detalhes:

Somador de base unária. Esta máquina determina o resultado de uma soma entre N números de forma unária.

Uso:

Término+Término=

Exemplos:

111+11=
1+1+1+1+1=

Restador

Detalhes:

Subtrador de base unária. Esta máquina determina o resultado da subtração entre DOIS números na base unária.

Uso:

'Término a'-'Término b'=
a != b ; a >= 1 ; b >= 1

Exemplos:

111-11=
11-111=

Duplicador

Detalhes:

Duplicador de correntes. Esta máquina duplica uma corrente de símbolos a, b ou c de forma identica.

Exemplos:

aaabbbccc
ccaaaacab
cbacbacba

Palíndromos

Detalhes:

Verificador de palíndromo. Esta máquina verifica se uma corrente é um palíndromo. Se a fita está vazia, a corrente é um palíndromo.

Exemplos:

aaabbbaaa
ccaaaacc
cbacbcabc

Multiplicador

Detalhes:

Multiplicador de base unária. Esta máquina determina o resultado da multiplicacão entre DOIS números na base unária.

Uso:

'Término a'*'Término b'=
a >= 1 ; b >= 1

Exemplos:

111*11=
11*111=
